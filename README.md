### Shortcuts

| Key | Do |
| --- | --- |
| <kbd>SHIFT</kbd> <kbd>\{ \( \<</kbd> | Wrap selection |
| <kbd>CTRL</kbd> <kbd>\[ \]</kbd> | Indent selection |
| <kbd>CTRL[SHIFT]</kbd> <kbd>Enter</kbd> | Insert line below or [SHIFT]above |
| <kbd>F5</kbd> | Reload App |
